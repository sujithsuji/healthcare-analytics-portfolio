# 🏥 Healthcare Outcomes & Operational Efficiency Analytics
Healthcare Outcomes &amp; Operational Efficiency Analytics using Python, SQL and Power BI
## Overview
End-to-end healthcare analytics project analyzing 71,518 patient 
records to evaluate hospital readmissions, length of stay, and 
operational efficiency.

## Key Results
- Overall Readmission Rate: 39.9%
- Average Length of Stay: 4.29 days
- Average Medications per Patient: 15.71
- Patients aged 70-80 had highest readmission rate at 43.9%

## Tools Used
| Tool | Purpose |
|------|---------|
| Python (Pandas, NumPy, SciPy) | Data cleaning & analysis |
| SQL (SQLite) | Data modeling & querying |
| Power BI | Interactive dashboard |
| Google Colab | Development environment |

## Dataset
Source: Diabetes 130-US Hospitals (1999-2008)
Records: 71,518 unique patient encounters

## Key Findings
- Patients aged 70-80 had the highest readmission rate
- Higher diagnosis count leads to longer hospital stays
- Readmitted patients had significantly more medications
- Majority of patients stayed 1-4 days in hospital

## Project Files

- dashboard/ — Power BI and HTML dashboard
